Preface

Reader, welcome to Shepherd's Library. If you are reading this text, I assume you already have some preliminary understanding of this library. Therefore, I won't elaborate further here. The only thing to emphasize is: don't get lost in the library. To prevent this from happening, please remember that no matter what, you can visit the Shepherd's Library Navigator Hub (usually found at the bottom of each page) to gain some understanding of where you are within the library. You can also directly access pages that interest you from that hub. With that, all preparations should be complete. Reader, begin your journey of acquiring knowledge.

Introduction

This page can be considered the main page of Shepherd's Library. Please do not confuse it with the Shepherd's Library Home. The latter is merely the homepage of the "Shepherd's Library Project," whereas this page is the actual "Shepherd's Library" page. This does not imply that either page is more important — rather, it simply distinguishes them. The Shepherd's Library Home is equally important, and you should have visited it before arriving here.

We categorize all knowledge in the library into several major types. Considering that a vast amount of knowledge will eventually be stored here, we plan to add a dedicated page in the future for storing other pages. However, in the early stages, I intend to focus primarily on addressing structural issues during updates, which are equally important. Therefore, in the initial phase, the library will contain very little recorded knowledge. Nevertheless, to better handle structural issues, I will appropriately include some knowledge during this period as a temporary solution — such as the articles you see on this page.
