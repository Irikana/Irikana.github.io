Basic Features

Basic features are those that any library should possess, and Shepherd's Library is no exception.

Icons

In the library, every page has an icon. Icons help you quickly identify the type or topic of a page.

Icons are an attribute of pages in Notion, displayed to the left of link buttons when a page is referenced, serving as a quick identification aid. Icons can be emojis or custom-uploaded images.

Note

If you see golden icons at the very top of some pages, or when they are referenced, it means they serve structural purposes beyond regular content.

Advanced FeaturesDeprecated

Advanced features build upon basic features through additional arrangements, presenting relatively more complex functionality. Below are advanced features supported in Notion:

NestingDeprecated

Supports containing multiple pieces of content within a single page, such as having multiple H1 headings, multiple H2 headings, etc.

Nesting refers to placing another collapsible block inside a collapsible block, forming a hierarchical structure. This approach suits displaying content with layered relationships, such as tables of contents or category descriptions.

A Callout is an information box with an icon and background color, used to highlight important content or notices. Callouts can include titles and body text, with icons being emojis or custom images.

On the website, nesting is implemented through page hierarchy structure.

Website Features

Beyond the library's internal features, the website itself provides some features accessible from anywhere, without being limited by library content.

Collapsible Sections

Allows users to collapse or expand certain content to improve reading experience. This feature is widely used across the website. The block you're currently viewing is an example of a collapsible section.

Click to expand and view sample content

This is collapsed content. On the Introduction page, the encyclopedia entries for SCP Foundation and The Wanderers' Library use this collapsible format to organize information.

Quick Navigator

A floating navigation bar on the left side of the page that expands on hover to show the current page's table of contents and quick links. On mobile devices, it switches to tap-to-expand mode.
