# ShepherdsLibrary (牧羊人图书馆)

这是公开的可供外人查阅的牧羊人图书馆的项目，该项目创建于2026年1月6日晚上23点左右。

## 项目简介

牧羊人图书馆（Shepherd's Library）是一个存放所有知识之地的个人知识库网站。亦被称为谢沷尔德图书馆（Shpherd's Library），任何一个可以实现该行为的事物都可以称为牧羊人图书馆。

### 核心理念

- **存放所有知识之地** - 作为个人知识管理和展示的平台
- **分馆制度** - 采用主馆和分馆的架构，便于知识的分类管理
- **持续建设** - 项目处于Alpha开发阶段，不断完善中

## 主要模块

### 1. 图书馆主馆 (Main Library)

**定位**: 核心知识和规则的存放地

包含以下核心页面：
- **图书馆入门** (`library/intro.html`) - 介绍牧羊人图书馆的概念、背景和目标
- **图书馆规则** (`library/rule.html`) - 定义图书馆的使用规范和行为准则
- **图书馆功能** (`library/feature.html`) - 说明图书馆具备的能力和功能特性
- **文章库** (`library/paper/`) - 存放各类文章和记录

### 2. 知识馆 (Knowledge Hall)

**定位**: 分馆，专门用于系统化知识管理

知识馆于2026年5月10日正式启用，自2026年3月2日起雏形已完成。知识馆将知识分为三大类：

1. **现象** (Phenomenon) - 观察到的现象和事实
2. **可追溯知识** (Traceable Knowledge) - 有明确来源和依据的知识
3. **可回忆知识** (Recallable Knowledge) - 基于经验和记忆的知识

**设计特点**:
- 整洁、简约的设计风格
- 文字醒目，适当增大字体
- 侧边栏导航（无顶部导航栏）
- 包含三种知识分类入口

### 3. 导航系统

- **主页轮播新闻** - 展示最新动态和更新
- **导航枢纽** (`navigator.html`) - 全站导航中心
- **便携式导航仪** - 响应式移动端导航
- **快速导航浮窗** - 桌面端快捷导航

## 技术特点

### 前端技术
- **纯静态网站** - HTML + CSS + JavaScript
- **语义化HTML** - 使用 `<header>`, `<nav>`, `<main>`, `<footer>` 等标签
- **CSS变量系统** - 统一管理颜色、间距等设计令牌
- **响应式设计** - 支持桌面端、平板和手机端
- **暗色模式支持** - 适配 `prefers-color-scheme: dark`

### 设计风格
- 简约线条风（自2026年4月更新起采用）
- 黄色主题配色
- 清晰的信息层级
- 轮播新闻展示

### 功能特性
- 新闻轮播组件（支持键盘导航）
- 快速回到顶部按钮
- 浮动知识馆入口
- 移动端适配导航

## 如何使用

### 本地预览
由于是纯静态网站，可以直接在浏览器中打开 `index.html` 文件进行预览。

### 在线访问
项目部署在 GitHub Pages 上，可通过仓库地址直接访问。

## 开发规范

项目遵循严格的开发规范，详见 [.trae/rules/project_rules.md](.trae/rules/project_rules.md)

核心原则包括：
- **文字与图片内容保护** - 不随意修改原有内容
- **设计确认机制** - 不明确时主动确认
- **红色警告框使用原则** - 仅在必要时使用警示框
- **更新日志检查流程** - 每次开发前必须检查更新日志
- **代码规范** - 语义化HTML、CSS变量、响应式设计
