视觉组件标准

本文档定义了项目中所有 UI 组件的标准样式。开发新页面或修改现有页面时，应直接复制对应组件的样式代码，以确保全站视觉一致性。

每个组件下方展示「实际渲染效果」和「对应代码」，方便直观对比参考。

1. 信息提示框（Callout Boxes）

蓝色信息框 — .function-box-blue

用途：一般性信息提示、功能说明、补充说明、"暂无条目"等中性通知。默认首选的信息框类型。

实际效果：

此处填写信息内容。

HTML 代码：

<div class="function-box-blue">
  此处填写信息内容。
</div>

灰色引用框 — .quote-box-grey

用途：引用性说明、参考来源说明、补充背景资料。

实际效果：

此处为引用或参考性文字内容。

HTML 代码：

<div class="quote-box-grey">
  此处为引用或参考性文字内容。
</div>

红色警告框 — .notice-box-red

用途：仅用于真正需要警示的内容（Alpha/Beta 阶段重要建设状态提醒、内容时效性警示、需要读者特别注意的约束条件）。未经作者明确许可不得使用。

实际效果：

此处为警告/警示内容。

HTML 代码：

<div class="notice-box-red">
  此处为警告/警示内容。
</div>

通用 Callout — .callout

用途：带图标的灵活提示框（Notion 风格），支持图标+文本并排布局。

实际效果：

HTML 代码：

<figure class="callout block-color-gray_background">
  <span class="icon">ℹ️</span>
  <div><p>此处为 callout 提示内容。</p></div>
</figure>

2. 折叠块 — <details>

用途：可展开/收起的长内容区域，如详细说明、扩展阅读等。

基本折叠块（默认折叠）— 实际效果：

点击展开的标题

折叠内容的正文放在这里。

已展开的折叠块 — 实际效果：

默认展开的标题

这段内容默认就是展开状态，用户可以直接阅读。

嵌套折叠块 — 实际效果：

外层标题

外层内容。

内层标题

内层内容。

HTML 代码（基本折叠块）：

<details>
  <summary>点击展开的标题</summary>
  <div>
    <p>折叠内容的正文放在这里。</p>
  </div>
</details>

<!-- 已展开 -->
<details open>
  <summary>默认展开的标题</summary>
  ...
</details>

<!-- 嵌套 -->
<details>
  <summary>外层标题</summary>
  <div>
    <p>外层内容。</p>
    <details>
      <summary>内层标题</summary>
      <div><p>内层内容。</p></div>
    </details>
  </div>
</details>

4. 新闻卡片

类型一：文字新闻（无海报）

类名：.news-card-text-only，用于没有海报图片的文字类新闻。

实际效果：

知识馆正式启用

2026年5月10日

点击此处了解更多

HTML 代码：

<div class="news-carousel-card" data-date="YYYY-MM-DD">
  <div class="news-card">
    <a href="{链接}" target="_blank" class="news-card-text-only">
      <h3 class="news-card-title">{标题}</h3>
      <p class="news-card-date">{YYYY年M月D日}</p>
      <p class="news-card-hint">点击此处了解更多</p>
    </a>
  </div>
</div>

类型二：海报新闻（有海报图片）

类名：.news-card-content，用于带有海报图片的新闻。

实际效果：

SOTM I 2026后记

2026年2月3日

点击海报了解更多

HTML 代码：

<div class="news-carousel-card" data-date="YYYY-MM-DD">
  <div class="news-card">
    <div class="news-card-content">
      <a href="{链接}" target="_blank" class="news-card-image">
        <img src="./image/poster/{路径}/{文件名}.png" alt="{描述}">
      </a>
      <div class="news-card-info">
        <h3 class="news-card-title">{标题}</h3>
        <p class="news-card-date">{YYYY年M月D日}</p>
        <p class="news-card-hint">点击海报了解更多</p>
      </div>
    </div>
  </div>
</div>

5. 浮窗

浮窗在正式页面中使用 position: fixed 定位。以下用相对定位模拟其外观效果。

知识馆浮窗 — .knowledge-hall-float

用途：主页右侧悬浮的知识馆入口浮窗。

外观模拟（relative 定位）：

知识馆

牧羊人图书馆分馆，存放和查阅知识之地

进入知识馆

HTML 结构：

<div class="knowledge-hall-float" id="kh-float">
  <div class="kh-float-header">
    <span class="kh-float-title">知识馆</span>
    <button class="kh-float-close"
      onclick="document.getElementById('kh-float').classList.add('hidden')">
      &times;
    </button>
  </div>
  <div class="kh-float-body">
    <p class="kh-float-desc">牧羊人图书馆分馆，存放和查阅知识之地</p>
    <a href="./knowledge-hall/index.html" class="kh-float-link">进入知识馆</a>
  </div>
</div>

语言切换浮窗 — .lang-switch-float

结构与知识馆浮窗类似，位置在知识馆浮窗下方 140px 处。

外观模拟（relative 定位）：

语言 / Language

切换网站显示语言

中文版 English

HTML 结构：

<div class="lang-switch-float" id="lang-float">
  <div class="lang-float-header">
    <span class="lang-float-title">语言 / Language</span>
    <button class="lang-float-close">&times;</button>
  </div>
  <div class="lang-float-body">
    <p class="lang-float-desc">切换网站显示语言</p>
    <a href="../index.html" class="lang-float-link">中文版</a>
    <a href="../en/index.html" class="lang-float-link">English</a>
  </div>
</div>

6. 导航组件

快速导航项 — .quick-nav-item

便携式导航仪中的导航项，悬停时高亮并右移。

实际效果：

图书馆主页 图书馆入门 导航枢纽

HTML 代码：

<a href="{路径}" class="quick-nav-item">导航项文字 </a>

页面跳转按钮 — .link-to-page

Notion 风格的内联跳转按钮。

实际效果：

牧羊人图书馆主页

HTML 代码：

<a href="{路径}" class="link-to-page">页面名称 </a>

外部/跨页链接 — .ext-link

标识"离开当前页面"的外部链接或跨页面链接，自动添加箭头图标。

实际效果：

外部链接示例

HTML 代码：

<a href="{URL}" class="ext-link">链接文本 </a>

7. 按钮 & 链接

浮动按钮 — .float-button

回到顶部、导航枢纽等固定位置的圆形按钮。正式页面中为 position: fixed，此处用相对定位模拟。

外观模拟：

HTML 代码：

<button title="回到顶部" class="float-button back-to-top"
  onclick="window.scrollTo({top: 0, behavior: 'smooth'})">
  回到<br>顶部
</button>

<button title="导航枢纽" class="float-button nav-hub"
  onclick="openNavigator()">
  导航<br>枢纽
</button>

已废弃标记 — .deprecated-badge

标记已废弃的内容。

实际效果：

此功能已废弃

HTML 代码：

<span class="deprecated-badge">已废弃</span>
