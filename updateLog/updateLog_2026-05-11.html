主馆变更

响应式设计与移动端优化 [Alpha 008]

- 移动端导航栏（.mobile-nav）现在在768px以下屏幕正确显示，之前始终隐藏

- body 在移动端添加 padding-bottom 为底部导航栏留出空间

- 底部导航栏支持 safe-area-inset-bottom 适配刘海屏手机

- 浮动元素（知识馆浮窗、语言切换浮窗）在移动端改为全宽自适应布局，避免溢出屏幕

- 页脚在移动端增加额外底部间距，避免被固定导航栏遮挡

- 新闻轮播添加触摸滑动支持（touchstart/touchmove/touchend），支持左右滑动切换新闻

- 轮播容器添加 touch-action: pan-y pinch-zoom 防止滑动冲突

- 英文版主页同步添加轮播触摸滑动支持

交互体验优化 [Alpha 008]

- 所有交互元素添加 -webkit-tap-highlight-color: transparent 移除移动端点击高亮

- .mobile-nav-link 增大内边距提升触摸目标大小，添加 :active 缩放反馈

- .float-button 添加 :active 缩放反馈

- .news-card 添加 :active 边框高亮反馈

- .kh-nav-item 添加 :active 背景反馈

- .kh-card 添加 :active 边框高亮反馈和过渡动画

- .mobile-nav 添加滑入/滑出过渡动画

CSS 移动端细节优化 [Alpha 008]

- 新增 375px 超小屏幕断点的细化适配（导航栏、浮窗、标题字号等）

- .content-bg-white-sub 在移动端优化内边距

- .section-title-text-sub / .section-title-text-main 在移动端缩小字号

- .subsection-header / .sub-subsection-header 在移动端缩小字号和间距

- .subsection-content 在移动端缩小内边距和字号

- .callout 在移动端改为纵向排列

- details/summary 在移动端缩小字号和内边距

- .link-to-page / .deprecated-badge 在移动端缩小字号

- .desc-font / .title-text 在移动端缩小字号

- body 添加 -webkit-text-size-adjust: 100% 防止 iOS 自动调整字号

知识馆变更

知识馆移动端侧边栏重构 [Alpha 008]

- 知识馆4个页面（index.html、phenomenon.html、recallable.html、traceable.html）侧边栏在移动端改为抽屉式覆盖层

- 新增汉堡菜单按钮（.kh-sidebar-toggle），三横线动画变形为叉号

- 新增遮罩层（.kh-sidebar-overlay），点击遮罩可关闭侧边栏

- 侧边栏打开时禁止背景滚动（body overflow: hidden）

- 支持 ESC 键关闭侧边栏

- 侧边栏添加 transform 过渡动画（0.3s cubic-bezier）

- 汉堡菜单按钮在全局 CSS 中定义，仅在 768px 以下显示

知识馆卡片与内容移动端优化 [Alpha 008]

- .kh-card 在移动端缩小内边距和间距

- .kh-card-title-link h3 在移动端缩小字号

- .kh-footer 在移动端缩小上间距

- 新增 375px 超小屏幕断点：进一步缩小内容区间距和字号

- .kh-main 在移动端增加顶部内边距为汉堡菜单留出空间

- 暗色模式下遮罩层背景色加深（rgba(0,0,0,0.5)）
