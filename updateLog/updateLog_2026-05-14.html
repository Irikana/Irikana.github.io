主馆变更

全站性能与体验优化（不改变内容和风格）[Alpha 014]

- 为所有页面添加 CSS 预加载（<link rel="preload">），加速关键资源加载

- 为所有页面添加 Open Graph meta 标签（og:title, og:description, og:type, og:locale），优化社交媒体分享显示

- 为所有页面添加 theme-color meta 标签，统一移动端浏览器主题色为 #2c3e50

- 为所有页面添加 referrer policy meta 标签（origin-when-cross-origin），增强隐私保护

- 为所有页面添加 Skip Navigation 无障碍链接（skip-navigation），支持键盘用户快速跳转到主要内容

- 为所有 main 标签添加 id="main-content"，配合 skip-navigation 使用

- 为所有 img 标签添加 width/height 属性，避免布局偏移（CLS）提升 Core Web Vitals

- 为非首屏图片添加 loading="lazy" 懒加载属性，减少首屏加载时间

- 为所有 target="_blank" 的外部链接添加 rel="noopener noreferrer" 安全属性，防止安全漏洞

- 在 css/style.css 中添加 .skip-navigation 样式（含暗色模式支持）

- 在 knowledge-hall/index.html 内联样式中添加 .skip-navigation 样式

  图书馆主馆：library/intro.html, library/rule.html, library/feature.html, library/library.html, library/visual-components.html
