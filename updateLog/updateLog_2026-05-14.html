主馆变更（深夜 - 动态功能系统 v1.0）

🚀 全站动态功能系统上线 [Alpha 015]

在不改变任何内容和风格的前提下，为网站添加7大动态交互功能：

1️⃣ 站内智能搜索

- 快捷键：Ctrl+K (Windows) / Cmd+K (Mac)

- 搜索范围：所有页面、文章、知识分类

- 支持中英文关键词匹配和结果高亮

- 结果分类显示：页面 / 分类 / 文章

- 即时响应，200ms 防抖优化

2️⃣ 自动目录导航 (Auto TOC)

- 自动扫描文章中的 h2/h3 标题生成目录

- 滚动时自动高亮当前章节

- 点击目录项平滑滚动到对应位置

- 支持折叠/展开，仅在有≥2个标题时启用

3️⃣ 字数统计与阅读时间

- 自动统计中文/英文字数

- 估算阅读时间（按500字/分钟计算）

- 显示在文章元数据区（article-meta）

4️⃣ 主题切换增强

- 三种模式：跟随系统 / 浅色 / 深色

- 点击按钮循环切换

- 使用 localStorage 记忆用户偏好

- 强制覆盖系统偏好设置

5️⃣ 本地书签/收藏夹

- 文章页面显示 ☆ 收藏按钮

- 点击收藏/取消收藏，状态持久化存储

- 右下角浮动按钮打开收藏面板

- 支持查看和管理所有收藏的文章

6️⃣ 阅读体验优化工具

- 字体大小调节：A- / A+ （80%-150%）

- 行宽模式切换：窄(700px) / 标准 / 宽(1200px)

- 专注模式：隐藏所有干扰元素，沉浸式阅读

- 一键打印优化版

- 所有设置使用 localStorage 记忆

7️⃣ 多语言版本增强

- 自动检测浏览器语言偏好

- 语言不匹配时显示切换提示横幅

- 添加语言切换快捷入口

- 未翻译页面显示提示标记
